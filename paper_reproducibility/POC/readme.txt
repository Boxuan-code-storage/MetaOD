POC testbed is based 100 datasets from Emmott datasets [1]. For simplicity, an exerpt of datasets are provided in the dataset folder.

To execute the experiments, please run "demo_POC.py". 
To access the latest code, please visit http://tinyurl.com/metaod.

[1] Emmott, A., Das, S., Dietterich, T., Fern, A. and Wong, W.K., 2015. A meta-analysis of the anomaly detection problem. arXiv preprint arXiv:1503.01158.
